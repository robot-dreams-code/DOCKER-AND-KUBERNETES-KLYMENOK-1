Multi stage допоміг зменшити образ за рахунок того що ми використали образ з go для створення бінарника і після цього скопіювали і запустили його в чистому образі, так розмір зменшився з ~1 гігабайту до 10 мегабайт.

Також для того щоб зберегти образ на хабі потрібно було залогінитис, створити репозиторій, перейменувати імедж і запушити в репо.
```
docker login
docker tag
docker push
```
